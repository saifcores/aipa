# AIPA Backend (`aipa-be`)

Spring Boot 3.5 + Java 26 + LangChain4j payment operations assistant.

See the [root README](../README.md) for architecture, API usage, and Docker commands.

```bash
# from repo root
docker compose -f aipa-be/docker/docker-compose.yml up -d postgres

# from this directory
mvn spring-boot:run
```
